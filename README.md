# 舒雅开发套件（Suyaa）

基于.Net Standard 2.1的综合开发工具包，包括基本类型处理、ORM框架、日志管理等

Comprehensive development kit based on .Net Standard 2.1, including basic type processing, ORM framework, log management, etc
