# 舒雅开发套件（Suyaa）

多语言: [English](https://github.com/Suyaas/Suyaa/blob/main/README.md) | 简体中文

基于.Net Standard 2.1的综合开发工具包，包括基本类型处理、ORM框架、日志管理等
